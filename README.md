
## Clone project 

Clone github repo

```bash
  git clone https://github.com/dhyanivj/formik-form.git
  cd formik-form
```


## Run project 

Install my-project with npm

```bash
npm install
npm start
```
    
## Deployed URL

https://dhyanivj.github.io/formik-form/

## Build

To create production build

```bash
  npm run build
```


## Deployment

To deploy this project run

```bash
  npm run deploy
```
